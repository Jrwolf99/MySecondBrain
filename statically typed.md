# Statically Typed

A language is statically typed if the type of a variable is known at compile time.

see [[Dynamically typed]]

Languages that are statically typed:
* C
* C++
* [[Java]]
* Rust
* Go
* Scala


Languages that are dynamically typed:
- Perl
- Ruby
- Python
- PHP
- JavaScript
- Erlang
